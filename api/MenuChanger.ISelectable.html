GetISelectable(Neighbor)

Returns the most extreme ISelectable subelement in the specified direction.
For example, on a button, this would return itself. On a VerticalItemPanel, this would return the first item if called with Neighbor.Up.

SetNeighbor(Neighbor, ISelectable)

Sets the neighbor in the specified direction to be the given ISelectable.
Often used recursively: e.g. setting a panel as a neighbor of a panel calls SetNeighbor on ISelectable elements of each panel.
